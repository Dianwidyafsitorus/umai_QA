# umami_QA
# Cermati QA Automation Project

This repository contains an end-to-end UI automation test suite for validating key user flows on the umai website using Playwright framework.

---

## Tech Stack

- Playwright
- JavaScript (ES Modules)
- Playwright Assertions

---

## Project Structure
```text
umai_QA/
│
├── steps/ # Step definitions
├── pages/ # Page Object Model
```


## Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/Dianwidyafsitorus/umai_QA.git
cd umai_QA
```

### 2. Install Dependencies
```bash
npm install
npx playwright install
```

## Run Test Cases
### Run all feature
```bash
npm run test
npm run report
```
